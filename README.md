# `moments-frontend`

## `Getting started`

Clone this repo and cd into it

```
git clone https://github.com/dominicH247/moments-frontend
cd moments-frontend

```

Install the dependencies

```
npm i
```

Run the Server

```
expo start

```

Scan qr code to run app

Enjoy the app!

## `Built using`

-React Native
-AWS Amplify
-AWS Cognito
-Expo
-Lottie
